## âš™ How It Works

1.  [cite_start]*Registration:* Administrators register students and lecturers into a central database. [cite: 17]
2.  [cite_start]*Session Generation:* Unique QR codes are generated by the ESP32 CYD device for each lesson. [cite: 18, 28]
